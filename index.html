Design Lang
Beyond CSS

A declarative design language that compiles to CSS + JS. Instead of hacking CSS for modern UI patterns, Design Lang makes them native — physics, materials, depth, motion, accessibility, haptics, and more.

npm install -g @empire/design-lang
